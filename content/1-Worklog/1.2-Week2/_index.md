### Week 2 Objectives:

The goal of this week is to apply learned knowledge to **build and deploy** a simple application on AWS, including:
* Granting access to AWS services for the application via IAM Role.
* Practicing application integration with storage services (S3).
* Getting familiar with AWS databases (RDS/DynamoDB) for data storage and management.
* Completing the basic application deployment process from **EC2 → IAM → Storage → Database**.
