### Week 1 Objectives:

* Connect and network with members of First Cloud Journey.
* Learn about the office rules and procedures for going to the workplace.
* Learn how to structure and write Internship Report.
* Gain foundational knowledge of AWS services and practice using the Console and CLI.
* Build and configure VPC components (Subnets, Route Tables, IGW, Security Groups, NACLs).

* Learn how to clean up resources properly to avoid unnecessary charges.

### Week 1 Achievements:


* Connected with members of **First Cloud Journey** and understood internship policies, rules, and workplace procedures.
* Learned how to structure and write the Internship Report.
* Gained orientation with **AWS basics** and explored main service categories:

  * Compute
  * Storage
  * Networking
  * Database
* Successfully created and configured an **AWS Free Tier account**.
* Learned about **AWS Budgets** and explored **AWS Support** features.
* Managed access with **AWS IAM**, including:

  * Creating IAM users and groups
  * Learning basics of role-based access control
* Acquired fundamentals of **Amazon VPC** and studied **network security** using Security Groups and NACLs.
* Completed environment setup:

  * Created VPC & Subnet
  * Added Internet Gateway
  * Configured Route Table
  * Set up Security Group
  * Enabled VPC Flow Logs
* Studied **EC2 fundamentals**, including:

  * Instance types and their use cases
  * AMI (Amazon Machine Image)
  * EBS (Elastic Block Store) for persistent storage
* Practiced multiple **EC2 connectivity approaches** (SSH, Elastic IP, EC2 Instance Connect, Session Manager).
* Performed **hands-on EC2 exercises**:

  * Deployed instances from AMI
  * Attached and expanded EBS volume
  * Tested connectivity and basic operations
* Set up **CloudWatch monitoring & alerts**.
* Performed full **resource cleanup** to avoid unwanted charges: terminated EC2 instances, deleted NAT Gateway, released Elastic IPs, removed VPC endpoints, VPN resources, and all associated networking components.
