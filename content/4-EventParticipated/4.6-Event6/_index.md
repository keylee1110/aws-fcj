# Reflection Report: “AWS for SAP & Generative AI”
&emsp;**Event:** AWS for SAP Using GenAI.
&emsp;**Date:** Thursday, November 20, 2025.
&emsp;**Time:** 13:00 PM – 17:00 PM.
&emsp;**Location:** 26th Floor, Bitexco Tower, Ho Chi Minh City.

### Purpose of the Event.
- Explore how AWS supports SAP workloads.
- Learn about modernizing SAP with Generative AI capabilities.

### Key Contents.
- **SAP on AWS:** Migration strategies and benefits.
- **Amazon Q Developer:** Using AI coding assistants for SAP ABAP development.
- **Predictive Maintenance:** Integrating IoT and ML with SAP data.

### Key Takeaways.
- AWS provides strong support for SAP infrastructure.
- GenAI can significantly accelerate ABAP code development and refactoring.
- Modernizing legacy ERP systems is easier with cloud-native tools.
