# Bài thu hoạch “AWS Cloud Mastery Series #2: DevOps on AWS”
&emsp;**Sự kiện:** AWS Cloud Mastery Series #2.
&emsp;**Ngày:** Thứ 2, 17/11/2025.
&emsp;**Thời gian:** 8:30 AM – 17:00 PM.
&emsp;**Địa điểm:** Tầng 26, Bitexco Tower, TP.HCM.

### Mục Đích Sự Kiện.
- Làm chủ các thực hành DevOps sử dụng công cụ AWS.
- Tìm hiểu về quy trình CI/CD và Infrastructure as Code (IaC).

### Nội Dung Chính.
- **AWS CodeSuite:** CodeCommit, CodeBuild, CodeDeploy, CodePipeline.
- **Infrastructure as Code:** Đi sâu vào AWS CloudFormation và CDK.
- **Observability:** Giám sát với CloudWatch và X-Ray.

### Bài Học Rút Ra.
- Tự động hóa là trái tim của DevOps.
- CDK cho phép định nghĩa hạ tầng bằng ngôn ngữ lập trình quen thuộc.
- Triết lý "You build it, you run it".

### Trải Nghiệm.
- Workshop cả ngày với các bài lab chuyên sâu.
- Đã tự xây dựng thành công một pipeline CI/CD hoàn chỉnh vào cuối buổi.
