# Reflection Report: “AWS Cloud Mastery Series #2: DevOps on AWS”
&emsp;**Event:** AWS Cloud Mastery Series #2.
&emsp;**Date:** Monday, November 17, 2025.
&emsp;**Time:** 8:30 AM – 17:00 PM.
&emsp;**Location:** 26th Floor, Bitexco Tower, Ho Chi Minh City.

### Purpose of the Event.
- Master DevOps practices using AWS tools.
- Learn about CI/CD pipelines and Infrastructure as Code.

### Key Contents.
- **AWS CodeSuite:** CodeCommit, CodeBuild, CodeDeploy, CodePipeline.
- **Infrastructure as Code:** Deep dive into AWS CloudFormation and CDK.
- **Observability:** Monitoring with CloudWatch and X-Ray.

### Key Takeaways.
- Automation is the heart of DevOps.
- CDK allows defining infrastructure using familiar programming languages (TypeScript/Python).
- "You build it, you run it" philosophy.

### Event Experience.
- Full-day workshop with intensive hands-on labs.
- Successfully built a fully automated CI/CD pipeline by the end of the day.
