# Reflection Report: “AWS Cloud Mastery Series #3: Security on AWS”
&emsp;**Event:** AWS Cloud Mastery Series #3.
&emsp;**Date:** Saturday, November 29, 2025.
&emsp;**Time:** 8:30 AM – 12:00 PM.
&emsp;**Location:** 26th Floor, Bitexco Tower, Ho Chi Minh City.

### Purpose of the Event.
- Understand the Shared Responsibility Model.
- Learn how to secure workloads on AWS.

### Key Contents.
- **IAM Best Practices:** Least privilege, MFA, Roles vs Users.
- **Network Security:** VPC, Security Groups, NACLs, WAF.
- **Data Protection:** Encryption at rest (KMS) and in transit.
- **Threat Detection:** GuardDuty and Security Hub.

### Key Takeaways.
- Security is "Job Zero" at AWS.
- Always follow the Principle of Least Privilege.
- Automate security checks wherever possible.

### Event Experience.
- Eye-opening session on common security misconfigurations.
- The "Capture the Flag" (CTF) style mini-game was very engaging.
